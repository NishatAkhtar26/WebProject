# React and Animation Project

This is a React web application built using Vite and Tailwind CSS. 
It demonstrates smooth UI animations, reusable components, and responsive design using modern frontend development practices.

## Features
- React + Vite setup for fast development
- Component-based architecture
- Smooth UI animations
- Responsive design for all devices

## Technologies Used
- React
- Vite
- Tailwind CSS
- JavaScript
